## 源码分析

`baseInRange` 用来检测 `number` 是否在 `start` 和 `end` 的范围内。

源码如下：

```javascript
function baseInRange(number, start, end) {
  return number >= Math.min(start, end) && number < Math.max(start, end)
}
```

即 `number` 需要同时满足不小于 `start` 和 `end` 的较小值，不大于 `start` 和 `end` 的较大者。
