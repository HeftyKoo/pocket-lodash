## 源码分析

`flatMapDeep` 其实是 `flattenDeep` + `map` ，先用 `map` 得到一个新的集合，然后再压平，可以压平成只有一层。

源码如下：

```javascript
const INFINITY = 1 / 0
function flatMapDeep(collection, iteratee) {
  return baseFlatten(map(collection, iteratee), INFINITY)
}
```

可以看到 `baseFlatten` 传入的层级是 `INFINITY` 。
